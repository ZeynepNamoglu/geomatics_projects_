# \### 1. Decimal Degree to DMS Converter (`convertion\_to\_degree.py`)

# A lightweight, interactive Python utility that converts raw decimal degree coordinates (often from GPS/GNSS logs) into the standard Degrees, Minutes, and Seconds (DMS) format. 

# 

# \*\*Features:\*\*

# \- \*\*Interactive CLI:\*\* Accepts real-time user inputs for coordinates and data types (Latitude/Longitude).

# \- \*\*Directional Awareness:\*\* Automatically calculates and appends correct hemispheric directions (North, South, East, West) based on the coordinate's sign.

# \- \*\*Robust Math:\*\* Handles negative coordinates flawlessly using absolute values and rounds seconds to two decimal places for GIS accuracy.

# 

# \*\*Usage:\*\*

# ```bash

# python convertion\_to\_degree.py
